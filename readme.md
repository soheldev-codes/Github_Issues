1️- Difference between var, let, and const

var – old way. Can be changed and redeclared. Function-scoped.

let – modern way. Can be changed but cannot redeclare in same block. Block-scoped.

const – cannot be changed after setting. Block-scoped. Used for values that stay the same.

2️- What is the spread operator ...

... spreads out values from an array or object.

Useful for combining arrays, copying arrays, or updating objects without changing the original.

3️- Difference between map(), filter(), and forEach()

map() – creates a new array by changing each item.

filter() – creates a new array but only keeps items that match a condition.

forEach() – loops over items and does something with them, but does not create a new array.

4️- What is an arrow function?

Short way to write a function using =>.

Can skip parentheses if only one argument.

Automatically returns a value if only one expression.

5️- What are template literals?

Strings written with backticks ` instead of quotes.

Can insert variables directly with ${}.

Can write multi-line strings easily.
